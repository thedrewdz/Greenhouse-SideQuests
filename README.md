# Greenhouse-SideQuests

A place for all the experiments. Each top-level directory is a self-contained spike or experiment with its own harness and README.

## Projects

| Project | What it is |
|---|---|
| [Temp-Test](Temp-Test/) | ESP32 firmware reading a temperature/humidity sensor and posting readings to an Azure Function endpoint. |
| [TempTestServices](TempTestServices/) | Test-oriented .NET workspace (Azure Functions host + application/domain/infrastructure layers) for Greenhouse service experiments. |
| [flutter_pi_spike](flutter_pi_spike/) | Minimal Flutter app proving we can build and deploy a `flutter-pi` application on a Raspberry Pi 4 (full-screen render + touch). |
